# Brebis brigasque (site statique)

Ce dépôt contient une exportation statique (HTML/CSS/JS) du site.

## Structure
- `site/` : pages HTML (point d’entrée : `site/index.html`)
- `assets/` : CSS/JS/polices, etc.
- `media/` : images utilisées

## Utilisation
Ouvre `site/index.html` dans un navigateur.
